## ## How To Use

Install plugin in your project with npm:
https://www.npmjs.com/package/p14-popup 

```bash
  npm i p14-popup
```

To install storybook:
```bash
  npx storybook init
```

In the project directory, you can run:
```bash
  npm run storybook
```
